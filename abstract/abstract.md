## Создание нового проекта
Создание нового проекта производится из командрой строки. Рекомендуется перед началом проекта инициализировать репозиторий `git init`, после чего скачать или создать файл `.gitignore` для проектов на C#. Если проект предусматривает создание нескольких программ, то создается папка, и проект на C# создается в этой папке (в терминале необходимо перейти в нее) командой `dotnet new console`. При этом работа с Git производится в корневом каталоге репозитория. Рекомендуется создать в VSCode отдельное окно для работы с проектом и отдельное для работы с Git.

Только что созанный проект C# содержит программу "Hello world!". Для того, чтобы убетится в том, что проект создан успешно, .NET работет корректно, необходимо выполинить в папке  проекта команду `dotnet run`. На экране появится приветствие этму бренному миру.

Запуск консольных программ, написанных на C# осуществляется командой `dotnet run`, выполненной в папке проекта.

## Типы данных
### Целочисленные
#### sbyte
8 бит. Значения от -128 до 127
#### byte
8 бит. Значения от 0 до 255
#### short
16 бит. Значения от -32 768 до 32 767
#### ushort
16 бит. Значения от 0 до 65 535
#### int
32 бита. Значения от -2 147 483 648 до 2 147 483 647
#### uint
32 бита. Значения от 0 до 4 294 967 295
#### long
64 бита. Значения от -9 223 372 036 854 775 808 до 9 223 372 036 854 775 807
#### ulong
64 бита. Значения от 0 до 18 446 744 073 709 551 615
### С плавающей запятой
#### float
64 бита. Значения от ±1,5x10^−45^ до ±3,4x10^38^
#### double
128 бит. Значения от ±5,0×10^−324^ до ±1,7×10^308^
#### decimal
256 бит. Значения от ±1,0x10^-28^ до ±7,9228x10^28^
В отличие от `double` тип `decimal` подходит, если требуемая степень точности определяется числом цифр справа от десятичной запятой. Такие числа обычно используются в финансовых приложениях для денежных сумм.

### Строки

Строки задаются типом `string`. Максимальный размер строки в C# составляет 2 Гб (это более 1 миллиона символов в кодировке Unicode). В памяти компьютера строка хранится как массив последовательных символов, каждый из которых имеет свой индекс и к нему можно обратиться. Например:

    string numberString = "0123456789";
    Console.WriteLine(numberString[4]);

Результатом будет вывод в терминал `4`.

### Логический тип

Логический тип `bool` может принимать всего два значения: `true` (правда, 1) или `fasle` (ложь, 0).

### Преобразования типов
#### Неявные преобразования
Некоторые типы поддерживают неявные преобразования (исходный тип: кому):
* sbyte:    short, int, long, float, double, decimal или nint
* byte:     short, ushort, int, uint, long, ulong, float, double, decimal, nint или nuint
* short:    int, long, float, double или decimal либо nint
* ushort:   int, uint, long, ulong, float, double или decimal, nint или nuint
* int:      long, float, double или decimal, nint
* uint:     long, ulong, float, double или decimal либо nuint
* long:     float, doubleили decimal
* ulong:    float, doubleили decimal
* float:    double

Кроме того:
* Любой целочисленный тип неявно преобразуется в любой числовой тип с плавающей запятой.
* Значение константного выражения типа int (например, значение, представленное целочисленным литералом) может быть неявно преобразовано в sbyte, byte, short, ushort, uint, ulong, nint или nuint, если оно находится в диапазоне целевого типа.

#### Явные пеобзразования

Если неявное преобразование не поддерживается, возможно произвести явное преобразование типа. Это делается с помощью выражения приведения в формате (T)E, результатом которого значение E преобразуется в значение типа T: `int a = (int)x;`. При `double x = 1234.7;` значение переменной `a` будет равно `1234`.

#### Преобразование строк в числа

Преобразование строк осуществляется методом `Parse()`. Все примитивные типы имеют этот метод. В качестве аргумента `Parse()` принимает строку, а возвращает объект указанного типа. Например:

    int a = int.Parse("10");
    double b = double.Parse("23б3");

В результате переменная `a` получит значение `10`, а переменная b значение `23,3`.

## NULL в переменных C#

Переменным в C# можно разрешить или запретить иметь значение `null` (т. е. не иметь никаких данных). Для того, чтобы явно указать на то, что переменная не может иметь значение `null`, то при присваивании значения необходимо добавить оператор `!`. К примеру, `int a = 10!;`. В этом случае компилятор не будет выводить предупреждение о возможном `null` в переменной.

## Работа со строками

### Сложение строк

C# поддерживает соединение строк оператором `+`. Т. е. переменная `a` в результате операции `string a = "1" + "1";` получит значение `11`.

### Преобразование регистра ToLower(), ToUpper()

Преобразование символов в строке в нижний регистр осуществляется методом `string.ToLower()`. К примеру:

    string a = "ЭТО СТРОЧНЫЕ БУКВЫ".ToLower();
    Console.WriteLine (a);

Результат: это заглавные буквы

Аналогичный метод есть для перевода символов в верхний регистр:

    string a = "это прописные буквы".ToUpper();
    Console.WriteLine(a);

## Методы (функции)
### Console.Write (); Console.WriteLine ();

Функции `Console.Write ("строка");` и `Console.WriteLine ("строка");` выводят в терминал указанную строку `строка`. Разница между командами заключается в том что метод `Write` после вывода оставляет курсор на позиции после выведенной строки, а `WriteLine` помещает курсор на первую позицию новой строки.

### Console.ReadLine ();

Метод `ReadLine` считывает введенную пользователем строку в терминале. Например: `string userInput = Console.ReadLine();`. В результате переменной `userInput` будет присвоена строка, которую ввел пользователь.

### Random().Next(min,max)

Метод `Random().Next(min, max)` выдает целое число от min (включительно) до max (не включительно). Использование:

    int a = new Random ().Next (0, 9);

В результате переменной `a` будет присвоено произвольное значение от 0 до 8.

## Ветвления, условия

В C# ветвления осуществляются оператором if-else.
Пример:

    if (условие)
    {
        действие, когда условие выполняется
    } else
    {
        действие, когда условие не выполняется
    }

В условии могут быть использованы логические операторы И `&`, ИЛИ `|`, ИСКЛЮЧАЮЩЕЕ ИЛИ `^`, а также бинарные условные И `&&` и ИЛИ `||`. Бинарные условные операторы вычисляют правый операнд только если это необходимо.

Операция отрицания НЕ записывается знаком `!`.
